int main ()
1. Start
2. Call input ()
3. Call milesRun ()
4. Call displayMILES ()
5. End

input ()
1. Start
2. Read running duration in minutes, M and seconds, S
3. Read steps run in one minute, STEPS
4. Return

milesRun ()
1. Start
2. MILES = STEPS * 2.5 / 5280 * ( M + S/60)
3. Return

displayMILES ()
1. Start
2. Display MILES
3. Return
